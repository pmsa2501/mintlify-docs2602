## What this module is

The first thing to note: Module TOS-RNDC 1 is a web application, meaning it can be accessed from any device (laptop, desktop, tablet, or phone) as long as it has a web browser. Module TOS-RNDC 1 is used by Puerto de Mamonal to control and track cargo across an export-oriented flow that originates at the customer location. Cargo is registered in the system and then tracked end-to-end: passing through the PRC (Punto de Recepción y Control) outside the terminal, entering the terminal, moving through waiting and load/unload operations, and placing cargo in yards where it becomes part of inventory. From there, that inventory is used to load ship vessels for export.

A key element of the module is capturing and managing operational weights and events so they align with RNDC national database requirements (Registro Nacional de Despacho de Carga). TOS stands for Terminal Operating System, a standard industry term used by terminals to describe the software used to track and control day-to-day terminal operations. TOS-RNDC 1 is intended to be the first of multiple modules that feed operational data to RNDC over time (for example: future modules may cover import flows, internal terminal movements, or additional cargo types and workflows).

## What you will find in this documentation set

The module is comprised of the following elements (each one has a separate guide with the detailed steps):

- Access and Authentication: How users access the web app and sign in (Google account or email), including the dedicated login page and authentication flow details.
  - Note: A Demo Mode is available for users who don’t have an account or who want to explore the portal for training purposes (no authentication required).
- Profile Settings: Where users review and manage their personal account information (such as name, email, phone number, and ID) from the profile area in the sidebar.
- Menu Camiones: The truck-side operational flow, tracking cargo from origin through key steps leading up to and through terminal entry.
- Menu Buques: The inventory-to-vessel flow, covering how cargo in yards/inventory is managed and dispatched toward ship loading for export.
- Menu PRC (Turnos): The pre-terminal staging and metering area used to manage citas (appointments) and turnos (queue) to regulate truck flow into the terminal.
