## Feature Overview

The Download & Export system lets authorized users export data from the **Camiones** and **Buques** tables. Four export formats are available — Excel (XLSX), CSV, and two RNDC regulatory formats (REIN1 and REIN2). Files can be downloaded locally or sent directly to Box Drive.

The export dialog includes date range filtering, customizable column selection with grouped toggles, and automatic future-date prevention. RNDC formats use fixed columns per Colombian government specification.

---

## Export Formats

| Format | Purpose | Available For |
| --- | --- | --- |
| Excel (.XLSX) | Spreadsheet analysis, internal reporting | Camiones + Buques |
| .CSV | Data import, third-party systems | Camiones + Buques |
| .CSV (RNDC REIN2) | Regulatory submission — appointment/shift assignment (9 fixed columns) | Camiones only |
| .CSV (RNDC REIN1) | Regulatory submission — full entry/exit record (16 fixed columns) | Camiones only |

<Note>
  RNDC formats (REIN1 and REIN2) are only available for the Camiones table.
</Note>

## How to Export RNDC Files

<AccordionGroup>
  <Accordion title="Step 1 — Open the Export Dialog (Step 1 of 3)">
    From the Camiones table, click **Descargar** to open the export modal.
  </Accordion>

  <Accordion title="Step 2 — Select RNDC Format (Step 2 of 3)">
    Choose either **RNDC REIN2** (9 columns) or **RNDC REIN1** (16 columns). The column selector will disappear — RNDC columns are fixed per government specification.
  </Accordion>

  <Accordion title="Step 3 — Download or Send to Box (Step 3 of 3)">
    Select your destination and submit. The file generates as a TAB-separated flat file ready for upload to the RNDC portal.
    - URL for downloads for Camiones table: [https://pmsa.app.box.com/folder/357306486195](https://pmsa.app.box.com/folder/357306486195)
    <Tip>
      RNDC timestamp logic uses the initial weighing timestamp as the primary reference, with the appointment timestamp as fallback — per the RNDC bulk cargo specification.
    </Tip>
  </Accordion>
</AccordionGroup>

## Frequently Asked Questions

<AccordionGroup>
  <Accordion title="What export formats are available?">
    Excel (XLSX) and CSV are available for both Camiones and Buques. RNDC REIN1 and REIN2 are available for Camiones only.
  </Accordion>

  <Accordion title="Why can't I see the column selector?">
    If you selected an RNDC format, the column selector is hidden because those columns are fixed by government specification and cannot be customized.
  </Accordion>

  <Accordion title="Why are some dates grayed out in the calendar?">
    Future dates are automatically disabled to prevent selecting a range with no data.
  </Accordion>

  <Accordion title="Can I export to Box Drive more than once per day?">
    Currently, Box Drive exports use a filename based on the date and table name. If you try to send a second file on the same day for the same table, it may fail due to a duplicate-name conflict. Use local download as an alternative.
  </Accordion>

  <Accordion title="Where does the Box Drive file go?">
    Camiones and Buques exports are sent to separate, pre-configured Box folders. See your administrator for the exact folder paths.
    URL for Camiones table downloads: [https://pmsa.app.box.com/folder/357306486195](https://pmsa.app.box.com/folder/357306486195)
    URL for Buques table downloads: [https://pmsa.app.box.com/folder/357307014621](https://pmsa.app.box.com/folder/357307014621)
  </Accordion>

  <Accordion title="What date format is used in exported files?">
    Exported files follow the Colombian convention: DD/MM/YYYY hh:mm AM/PM.
  </Accordion>
</AccordionGroup>

## Troubleshooting

<AccordionGroup>
  <Accordion title="My Box export failed with a duplicate file error">
    This happens when you try to export to Box twice on the same day for the same table. The filename includes the current date, so a second export creates a name conflict. Use local download instead, or wait until the next day.
  </Accordion>

  <Accordion title="My export file is empty">
    Make sure the selected date range contains data. If the range is empty, the system will show an error message before generating the file.
  </Accordion>

  <Accordion title="I don't see the Descargar button">
    Verify that you have the correct role and permissions to access the export feature. Contact your administrator if the button is not visible.
  </Accordion>

  <Accordion title="The Box upload succeeded but I can't find the file">
    Check the configured Box folder for your table (Camiones and Buques use separate folders). If you are unsure of the folder path, contact your administrator.
  </Accordion>
</AccordionGroup>

#### Important Notes

<Tip>
  All exports run server-side and respect database-level access policies — file content never touches the client until download.
</Tip>

- RNDC REIN2 contains 9 fixed columns: Terminal NIT, transport company, vehicle plate, driver ID, manifest number, and offer timestamp.
- RNDC REIN1 contains 16 fixed columns: all REIN2 columns plus entry/exit timestamps, weights, and scale IDs.
- Exports include complete data for the selected range — there is no pagination cutoff.
- The system handles null values and negative numbers (inventory adjustments) correctly.
- Column selections reset each session — they are not persisted between visits.
